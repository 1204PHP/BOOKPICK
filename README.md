# BOOKPICK
## 주제: 도서추천 및 독서 커뮤니티
## 팀원: 오성찬, 여중기
## 프로젝트 기간: 2023년 12월 10일 - 2024년 1월 19일
## 링크 : http://112.222.157.156:6003
algolia api 관련하여 무료버전 사용으로 index설정이 초기화 되는 경우가 있습니다.<br>
재연동 시 실시간 연관검색어가 문제없이 작동가능합니다.
